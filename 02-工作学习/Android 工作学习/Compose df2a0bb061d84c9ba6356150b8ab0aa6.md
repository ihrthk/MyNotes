    # 二、用法
    
    ## 1、添加地图
    
    我们在`OmniMap`中提供了：`GDMap`，`TXMap`，`BDMap`，`HWMap`，`GGMap` 可组合项来渲染我们的**地图MapView**
    
    **渲染高德地图**
    
    ```
    复制代码GDMap(modifier = Modifier.fillMaxSize()){ ... }
    
    ```
    
    **渲染腾讯地图**
    
    ```
    复制代码TXMap(modifier = Modifier.fillMaxSize()){ ... }
    
    ```
    
    **渲染百度地图**
    
    ```
    复制代码BDMap(modifier = Modifier.fillMaxSize()){ ... }
    
    ```
    
    ## 2、配置地图
    
    在图可组合项，我们提供 `MapUiSettings`和`MapProperties` 配置地图
    
    > 
    > 
    > 
    > MapUiSettings：与UI相关的设置的数据类 MapProperties：可在地图上修改的属性的数据类
    > 
    
    [](Compose/512ab249e85b41208b6aff846d5aac39tplv-k3u1fbpfcp-zoom-in-crop-mark1512000.awebp)
    
    **高德地图**
    
    ```
    复制代码// 高德地图
    GDMap(
        modifier = Modifier.fillMaxSize(),
        properties = MapProperties(...),
        uiSettings = MapUiSettings(...)
    )
    
    ```
    
    [](Compose/79cae1b642fd4361885ecf634ae6ac0atplv-k3u1fbpfcp-zoom-in-crop-mark1512000.awebp)
    
    **腾讯地图**
    
    ```
    复制代码// 腾讯地图
    TXMap(
        modifier = Modifier.fillMaxSize(),
        properties = MapProperties(...),
        uiSettings = MapUiSettings(...)
    )
    
    ```
    
    ```
    复制代码// 百度地图
    BDMap(
        modifier = Modifier.fillMaxSize(),
        properties = MapProperties(...),
        uiSettings = MapUiSettings(...)
    )
    
    ```
    
    ## 3、移动地图相机
    
    在图可组合项，我们提供 `CameraPositionState` 来控制地图相机
    
    > 
    > 
    > 
    > CameraPositionState：控制和观察地图的相机状态
    > 
    
    [](Compose/0856b72e08d24bc7a80ffeb255d45425tplv-k3u1fbpfcp-zoom-in-crop-mark1512000.awebp)
    
    [](Compose/b0b51742686c4517ac2d47b55bc4bc60tplv-k3u1fbpfcp-zoom-in-crop-mark1512000.awebp)
    
    **高德地图**
    
    ```
    复制代码// 高德地图
    val cameraPosState = rememberCameraPositionState()
    val scope = rememberCoroutineScope()
    GDMap(
        modifier = Modifier.fillMaxSize(),
        // 默认提供的位置在：天安门
        cameraPositionState = cameraPosState
    )
    Button(onClick = {
        scope.launch {
            cameraPosState.animate(CameraUpdateFactory.newLatLngZoom(LatLng(40.186009,116.400088),13F))
        }
    }){
        Text(text = "移动地图相机")
    }
    
    ```
    
    **腾讯地图：**
    
    ```
    复制代码// 腾讯地图
    val cameraPosState = rememberCameraPositionState()
    val scope = rememberCoroutineScope()
    TXMap(
        modifier = Modifier.fillMaxSize(),
        // 默认提供的位置在：天安门
        cameraPositionState = cameraPosState
    )
    Button(onClick = {
        scope.launch {
            cameraPosState.animate(CameraUpdateFactory.newLatLngZoom(LatLng(40.186009,116.400088),13F))
        }
    }){
        Text(text = "移动地图相机")
    }
    
    ```

    # 5、自定义Marker覆盖物的InfoWindow
    
    提供2中修改`Marker`的`InfoWindow` :
    
    > 
    > 
    > 
    > `MarkerInfoWindowContent`：只填充子视图，不修改默认`InfoWindow`窗口背景 `MarkerInfoWindow`：完全定制`InfoWindow`窗口及内容
    > 
    
    ```
    复制代码// 只修改内容，不修改容器
    MarkerInfoWindowContent(
        //...
        title = "我是title",
        snippet = "我是snippet"
    ) { marker ->
        Column {
            // 如果是百度地图，请使用 marker.getTitleExt()
            Text(marker.title ?: "", color = Color.Green)
            // 如果是百度地图，请使用 marker.getSnippetExt()
            Text(marker.snippet ?: "", color = Color.Red)
        }
    }
    
    // 修改容器及内容
    MarkerInfoWindow(
        //...
        snippet = "我是一个卖报的小画家(自定义InfoWindow)"
    ) { marker ->
        Card(modifier = Modifier.requiredSizeIn(maxWidth = 88.dp, minHeight = 66.dp)) {
            Text(
                modifier = Modifier.padding(4.dp),
                // 如果是百度地图，请使用 marker.getSnippetExt()
                text = marker.snippet ?: "", color = Color.Red)
        }
    }
    
    ```
